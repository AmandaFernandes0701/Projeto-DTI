# Projeto-DTI
Em suma, o projeto realiza a seguinte ação: a partir dos dados fornecidos pelo usuário, ele checa qual  Pet Shop propõem o melhor custo benefício para o cliente a partir dos dados inseridos. Para a realização desse projeto, é válido salientar que eu atuei em duas frentes: - Frontend que pode ser executado no navegado
- Arquivo js que é rodado no próprio terminal
